# DevPulse — AI 辅助开发入口

> 基于 Next.js 14 的技术博客平台。

## 知识地图

AI 在此工程中按以下优先级加载上下文：

| 层级 | 位置 | 内容 | 加载时机 |
|------|------|------|---------|
| P0 | `CLAUDE.md` | 项目概述、技术栈、编码规范、常用命令 | **每次对话** |
| P1 | `.claude/rules/` | 各维度的核心约束（精简版） | **编写对应代码时自动加载** |
| P2 | `docs/` | 完整版规范、架构、设计决策 | **按需查找** |

## 目录索引

- `.claude/rules/` — AI 自动加载的硬约束（各文件≤15行）
- `docs/architecture/` — 系统架构、模块边界、数据流
- `docs/conventions/` — 完整版编码规范（含示例）
- `docs/design/` — 功能设计文档（feature-*.md）
- `docs/plans/` — 迭代计划和待办
- `docs/reference/` — 外部资料和自动生成文件

## 常用命令

| 命令 | 说明 |
|------|------|
| `/dev` | 启动开发服务器 |
| `/test <path>` | 为指定文件生成测试 |
| `/review <path>` | 审查代码质量 |
| `/commit` | 提交当前改动 |
| `/doc <path>` | 生成文档 |
| `/pr-review` | 审查当前分支 PR |
